# ANT 项目说明

## 项目简介

ANT 是一个智能体后端框架，集成了语音识别(ASR)、自然语言处理(NLP)和文本转语音(TTS)功能，着重于智能体个性化。项目名称 ANT 是由 ASR、NLP 和 TTS 三个核心模块的首字母组合而成。

## 核心功能

- **语音识别 (ASR)**：基于 Qwen3-ASR 模型，支持 52 种语言和方言的语音识别，包括中文、英语、日语、韩语等多种语言。
- **自然语言处理 (NLP)**：使用 OpenAI API 兼容实现，用于处理用户语音转文本后的自然语言理解和生成。
- **文本转语音 (TTS)**：基于 Qwen3-TTS 模型，支持多语言文本转语音，包括语音克隆、语音设计等功能。

## 安装方法

### 一键安装脚本

为了方便用户快速安装和配置 ANT 项目，我们提供了一键安装脚本，支持自动安装依赖并实现优雅降级：

```bash
# 运行安装脚本
python install.py

# 或使用命令行参数自定义安装
python install.py --help
```

安装脚本会自动：
1. 检测系统环境
2. 安装必要的依赖包
3. 尝试安装推荐的优化包（如 FlashAttention 2）
4. 当某些依赖安装失败时，会自动降级到功能兼容的版本
5. 验证安装结果

## 项目架构

ANT 项目采用模块化设计，各模块之间通过预留接口进行集成：

1. **ASR 模块**：负责将用户的语音输入转换为文本。
2. **NLP 模块**：负责处理转换后的文本，进行自然语言理解和生成。
3. **TTS 模块**：负责将 NLP 处理后的文本转换为语音输出。
